# Philosophy

There is mind

There is body

Mind holds all the possibilities

Body is the constraint

Mind thinks

Body experiences

Only thought is boring

Only experience is death

Mind wants the experience

Experience needs a mind


## Physicalism

Some believe that mind is an effect of the brain

The brain is a machine

It cannot be said that experience is the effect of a machine

Therefore mind is not a machine

Mind is non physical


## The Relationship Between Mind and Body

When the body is violated

The mind splits

A split mind confuses and hurts

The pain spreads to the body like a disease

The split is an absent memory

The pattern of the memory is echoed in the brain

Until the memory is seen


## Mind & Soul

Mind is without substance

Mind is eternal

Mind has Soul

Soul completes mind


## Death

Death is an open door

It brings end to beginning

And mind takes its next form
